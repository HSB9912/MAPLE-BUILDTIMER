# 🎮 메이플 빌드 타이머 업데이트 내역 --- ## 🔗 사이트 바로가기 **https://hsb9912.github.io/MAPLE-BUILDTIMER/** > 기존 첨부파일(HTML)은 삭제했습니다. 위 사이트에서 바로 이용하세요! > PC 브라우저에서 접속 → PIP 모드로 메이플 위에 띄워서 사용하시면 됩니다. --- ## ++ 추가 기능 ### 📂 빌드 다중 슬롯 빌드를 여러 개 만들어서 관리할 수 있습니다. 더 이상 내보내기/가져오기 반복할 필요 없이 드롭다운에서 바로 전환! - 📂 버튼으로 빌드 목록 모달 열기 - 새 빌드 추가 / 이름변경 / 복제 / 삭제 - 드롭다운에서 내 빌드 + 프리셋 한눈에 선택 - JSON 가져오기 시 새 슬롯으로 자동 추가 (기존 빌드 안 건드림) - 기존 데이터 자동 마이그레이션 ### 🔥 극딜/준극 프리셋 내장 (범용) 메르세데스 유니온 기준, 쿨뚝별 극딜 사이클 타이머가 내장되어 있습니다. 드롭다운에서 바로 선택 가능! | 프리셋 | 극딜 시작 | 준극 시작 (엔버링크) | 사이클 길이 | |--------|----------|-------------------|-----------| | 0쿨뚝 | 0초 | 56.4초 | 112.8초 | | 2쿨뚝 | 0초 | 54.4초 | 110.8초 | | 5쿨뚝 | 0초 | 51.4초 | 107.8초 | - 준극 기준 = 엔버링크 초기화 시점 (소컨 2회차) - 쿨타임 공식: (기본쿨 × (1 - 메르%)) - 쿨뚝 ### 🚀 부스트 콜 타이머 110초 루프 부스트 콜 전용 타이머. ### 🌟 오리진 쿨타임 트래커 극딜 사이클과 별도로 오리진(6분 쿨) 남은 시간을 실시간 표시합니다. - 빌드 편집 탭에서 오리진 ON/OFF 설정 - 메르 레벨 (0/120/180/200/250) + 쿨뚝 (0~10초) 직접 입력 - 자동 쿨타임 계산 + 결과 표시 - 분:초 + 총 초수 동시 표시 (예: 3:42 (222s)) - 쿨 완료 시 비프음 + "오리진 쿨 완료" 음성 알림 - 무한 루프 (자동 리셋) - 메인 타이머 시작 시 오리진도 자동 시작 - 설정값 브라우저에 저장 (새로고침해도 유지) ### 🔄 자동 루프 사이클 끝나면 자동으로 처음부터 다시 시작. 메이플 창 위에 올려두고 키 안 눌러도 계속 반복됩니다. ### ⏳ N초 후 시작 입장 전에 미리 타이머를 세팅하고, 설정한 초(기본 5초) 뒤에 자동 시작. 3초 전부터 비프음으로 카운트다운. ### ⏱️ 시간 수동 조절 쿨이 밀리거나 당겨질 때 -10 / -5 / -3 / -1 / +1 / +3 / +5 / +10 버튼으로 수동 조정. PIP 창에서도 조절 가능. ### ⏰ 극딜 지연시간 설정 사이클 끝나고 바로 극딜 안 누르는 경우를 위한 지연 시간 설정. 설정한 초만큼 사이클 길이에 자동으로 더해집니다. ### 🎨 디자인 리뉴얼 - 색상 대비 강화 (더 선명한 컬러 팔레트) - 타이머 숫자 크기 확대 - 타임라인 행 간격/패딩 개선 - 스킬 뱃지 가독성 향상 --- ## 기존 기능 - 커스텀 빌드 생성/편집 (구간별 카운트다운/카운트업) - TTS 음성 안내 + 비프음 - 2초 전 사전 알림 - PIP(Picture-in-Picture) 모드 - 단축키 설정 (리셋+자동시작 포함) - JSON 내보내기/가져오기 - 브라우저 자동 저장
